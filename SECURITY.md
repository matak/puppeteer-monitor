# Security Policy

## Supported Versions

We release patches for security vulnerabilities for the latest major version.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email **info@romanmatena.cz** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond as soon as possible and work with you to address the issue before any public disclosure.
